# ProjectPulse

**ProjectPulse** is a full-stack **MERN (MongoDB, Express, React, Node.js)** project designed to manage projects, tasks, and users in a team environment. It features role-based authentication, task tracking, project history, and admin/user dashboards.  

This project simulates a real-world project management application and showcases modern web development practices using **React, Tailwind CSS, Node.js, Express, and MongoDB**.

### Admin Features
- Create, view, and delete projects.
- Assign tasks to users within a project.
- Manage users (view and delete).
- Separate dashboards for admins and users.
- View project history including tasks and team members.

### User Features
- View tasks assigned to the user.
- Update task status (To-do, In-progress, Done).
- Stylish and responsive dashboard with Tailwind CSS.
- Logout functionality.

### General
- Role-based authentication (Admin/User).
- Responsive design using **Tailwind CSS**.
- Modular folder structure for clean code.
